 Overview

SRM Insider is a frontend-focused mobile application UI that reimagines club engagement through a clean, intuitive, and scalable design system.

This project emphasizes:

 User-centric UI/UX design
 Mobile-first layouts
 Component-based structure
 Real-world usability
 Screens Designed
Authentication – Login / Signup with clean, validation-ready UI
Home Dashboard – Announcements, domains, and recommended events
Events Module – Event browsing, filtering, and bookmarking UI
Notifications – Structured, real-time styled update cards
Profile – User info, activity stats, and settings interface
 Unique Feature (UI Concept)
 Idea Submission Interface

A dedicated UI flow that allows users to submit:

Event ideas
Project proposals

Designed to encourage innovation and active participation within the club ecosystem.

 Design Highlights
 Dark theme with gradient accents
Intuitive navigation (bottom navigation + FAB)
 Reusable and scalable UI components Consistent spacing, typography, and hierarchy
 Modern card-based layouts
 Tools Used
Figma – UI/UX Design
Android Studio – UI Implementation
Jetpack Compose – Layout Design
 Scope

This project is UI/Frontend only.
No backend, database, or API integration is included.

🚀 Future Scope
Backend integration (Firebase / REST APIs)
Chat / Query system
Resource sharing module
AI-based event recommendations
